# GitHub Guia básico - O perfil

Esse é um recurso óbvio, porém o que ele tem de poderoso ele tem de desconhecido, curiosamente ele é bem simples de implementar, tudo que você tem que fazer é [acessar esse link](https://github.com/settings/profile) ou clique em \"Edit profile\" e preencher o formulário do GitHub, abaixo um guia dos campos por ordem:

| Campo | O que é? |
|---|---|
| Name | É seu nome real |
| Public email |  É um email que você deixa para pessoas entrarem em contato direto com você, usualmente é seu \"contato comercial\" (nada de _mulekedugrau123@gmail.com_ coloque algo sóbrio) |
| Profile picture |  É a sua foto nada, de muito místico aqui |
| Bio |  É onde você pode se descrever |
| Pronouns |  É o seu pronome (meu caso he/him, ele/dele em inglês), coloque, por que não? |
| URL |  É o seu site pessoal, não tem um? Relaxa, vamos chegar nessa parte |
| Social accounts |  São os links para as suas redes sociais, um por campo |
| Company |  É a empresa que você trabalha/sua empresa |
| Location |  É o seu país/estado/cidade geralmente se coloca um emoji com a bandeira do país |
| Display current local time |  É o seu fuso horário o GitHub irá avisar a quem estiver acessano o quão adiantado ou atrasado a pessoa que acessou seu perfil está de você na maior parte do Brasil é \"GMT -3 - Brasilia\" |
| Location |  É o seu país/estado/cidade geralmente se coloca um emoji com a bandeira do país |
  
Ao chegar aqui clique em \"**Update profile**\" para salvar

| Campo | O que é? |
|---|---|
| Include private contributions on my profile | O GitHub possui um gráfico que ilustra as contribuições que você fez em projetos, por padrão apenas as contribuições públicas são contabilizadas, marcando essa opção as privadas também são contabilizadas (lembrando que isso **NÃO** torna repositórios públicos, apenas contabiliza suas contribuições) |
  
  Ao chegar aqui clique em \"**Update profile**\" para salvar

|---|---|

  Ao chegar aqui clique em \"**Save jobs profile**\" para salvar
  
-----------------------------------------------------------
 
| Campo | O que é? |
|---|---|

Tudo que foi mostrado é opcional, mas certamente deixa o perfil com um ar mais profissional, para prosseguir com a próxima sessão, abra seu perfil (clique em "Go to your personal profile" no topo da página)

## Customizando sua Overview - Básico

A primeira coisa que você pode fazer para customizar sua Overview é ir até \"Customize your pins\", na janela que abrir você poderá fixar até 6 itens (entre repositórios e notas) que julgar importante que as pessoas conheçam

## Customizando sua Overview - Avançado

Primeiramente inicie a [criação de um novo repositório](https://github.com/new), em **Repository name** coloque seu nome de usuário e você verá a seguinte mensagem:

> **usuario/usuario** is a ✨special ✨ repository that you can use to add a README.md to your GitHub profile. Make sure it’s public and initialize it with a **README** to get started.

Marque **Public** e principalmente **Add a README file** e por fim \"Create repository\"

Tudo que você fizer no arquivo `README.md` desse repositório irá aparecer na sua Overview, para prosseguir com a próxima sessão, abra seu perfil (clique no seu nome de usuário no topo esquerdo da página)
